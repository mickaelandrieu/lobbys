Scripts et éléments de scripts permettant de compléter l'onglet "Bilan Activités" du googleSheet "Obs_Lobby"

* Le dossier "listesValeurs" contient les valeurs des champs et des tables de correspondances, tels qu'ils apparaissent dans les données HATVP en septembre 2018.

* importHATVP.js est le script qui tourne sur le GS ci-dessous. Il comporte également des fonctions pour faire la liste (vérifier ?) des valeurs des champs HATVP (listes de listesValeurs)
	En pratique ce script tourne donc pour remplir "Essai Import HATVP" depuis le site de la HATVP en temps réel :
 	https://docs.google.com/spreadsheets/d/1uXWmrEqOkFXEEFnfd0mgsYgl6nljycvx6f7f6kU-nZE
	Les données ainsi obtenues ont été copiées/collées dans "ObsLobby/Bilan Activités" le 3 septembre 2018 afin de les trier.

* listeColonneOrdonnees est la liste ordonnée des colonnes du google sheet à remplir avec ce script.
